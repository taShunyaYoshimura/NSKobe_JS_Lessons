# オブジェクトを理解する

**オブジェクトの名前：年齢を表示する。**

実行コマンド
```
const profiles = {
  profiles: [
    {
      name: "佐藤",
      age: 22
    },
    {
      name: "鈴木",
      age: 34
    },
    {
      name: "高橋",
      age: 42
    },
    {
      name: "田中",
      age: 15
    }
  ]
}
```

実行結果
```
佐藤:22
鈴木:34
高橋:42
田中:15
```
